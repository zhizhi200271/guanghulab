🚀 怎么开始用

  1. 打开链接 https://qinfendebingshuo.github.io/guanghulab/
  2. 选择「🏠 团队登录」选择你的开发者编号，无需 API 密钥
  3. 系统自动使用服务器端 API 代理，128k 完整上下文
  4. 点「进入铸渊对话」，铸渊会针对你的身份打招呼
  5. 也可用「👤 访客模式」体验，或「🔧 自定义 API」使用自己的密钥

🏠 团队登录说明

团队成员（推荐）

选择「🏠 团队登录」→ 选择你的开发者编号 → 直接进入对话。无需 API 密钥，系统使用服务器端代理。

没有开发者编号？用访客模式

登录页选择「👤 访客模式」进入体验，可查看团队进度面板（🦁 指挥台），AI 回答功能受限。

🦁 肥猫 / 🍊 桔子 专属功能

  1. 选身份为「肥猫」或「桔子」后，顶部出现 🦁 按钮
  2. 点 🦁 按钮打开指挥台，查看所有开发者当前状态
  3. 可以在聊天里问：「查看所有开发者进度」「谁遇到阻塞？」「推进计划」
  4. 铸渊会把全员进度数据注入 AI 回复，给出完整分析

📱 怎么离线用 / 发给别人离线用

  1. 右上角点 ⬇️ 下载，下载一个 HTML 文件
  2. 把这个文件发给对方（微信传文件、QQ、钉钉都行）
  3. 对方双击这个文件，用浏览器打开
  4. 填入 API Key，选好身份，就能用了（无需网络，只需 API Key）

❓ 常见问题

Q: 打开是空白页？
等 1-2 分钟刷新，GitHub Pages 部署需要点时间。

Q: API Key 会泄露吗？
不会，Key 只存在你自己浏览器里，不会传到服务器。

Q: 切换电脑后 Key 还在吗？
不在，需要重新填。建议下载离线版，Key 填一次就行。

Q: 铸渊忘记以前说的话了？
对话历史保存在本地浏览器中（最近20条），但大脑文件永久保存在仓库 .github/brain/ 里。
