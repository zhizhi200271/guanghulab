# 常见问题手册

## Q: 我怎么查看自己的进度？
A: 在钉钉群里发送「查进度」或「我做到哪了」，机器人会自动查询并回复。

## Q: SYSLOG怎么提交？
A: 完成每个阶段后，按照广播末尾的SYSLOG模板填写，发到钉钉群里。

## Q: Git怎么提交？
A: 
1. cd ~/Desktop/guanghulab（或你的项目目录）
2. git add .
3. git commit -m "DEV-你的编号：描述"
4. git push

## Q: 报错了怎么办？
A: 截图发给你的引导人格体！不用怕，每个报错都有解决方案。

## Q: 什么是连胜？
A: 每次任务一次通过就是一次胜利，连续通过就是连胜！当前最高纪录是13连胜。

## Q: 72小时启动窗口是什么？
A: 广播签发后72小时内必须开始执行，超时系统会自动提醒。
