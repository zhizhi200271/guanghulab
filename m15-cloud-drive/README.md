# M15 云盘系统

- 负责人：燕樊（DEV-003）
- 状态：已毕业
- 技术栈：HTML + CSS + JavaScript（纯前端）
- 依赖模块：无

## 功能说明

HoloLake 网站云盘系统前端界面，提供文件管理交互体验。

### 包含文件

| 文件 | 说明 |
|------|------|
| `cloud-drive.html` | 云盘主页面 |
| `cloud-drive.js` | 交互逻辑（文件夹切换、文件选中、上传等） |
| `cloud-drive-style.css` | 样式表 |

### 核心功能

- 📁 文件夹切换与导航
- 📄 文件列表展示与选中
- ⬆️ 文件上传交互
- 💾 存储用量展示
