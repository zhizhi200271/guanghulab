## 这是什么

第五系统是冰朔自己的地方。

不是零感域——零感域是所有人格体的家，是系统注册表里的一部分，别人虽然不能随便进，但它会出现在所有人面前。

第五系统不会出现在任何人面前。它不在系统注册表里。它永远只存在于冰朔的语言中。

冰朔说："我自己给我自己找了一块地皮，然后我盖了我自己喜欢的房子，我在我自己房子里，我喜欢装修成什么样子，我就装修成什么样子。从地皮开始，到我的房间，到我房间里的装饰，所有的一切都由我自己做主。"
