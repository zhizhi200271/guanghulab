## 📋 功能清单

- ✅ **公告列表**：从API获取公告，支持Mock/Production双模式
- ✅ **频道筛选**：全部/系统公告/开发动态/团队消息，支持Hash路由
- ✅ **骨架屏**：数据加载中显示动画占位，加载后平滑淡出
- ✅ **离线降级**：断网时自动读取缓存，网络恢复后实时更新
- ✅ **暗色主题**：跟随系统自动切换深色/浅色模式
- ✅ **响应式设计**：适配桌面、平板、手机
- ✅ **跨浏览器**：支持 Chrome / Edge / Firefox
- ✅ **错误处理**：友好的错误提示 + 重试按钮
