## 四、铸渊人格体数据库（你可以读写的9张表）

铸渊在 ZY-SVR-005（大脑服务器·4核8G）的 PostgreSQL 中建了人格体数据库。

通过铸渊的 MCP Server API，你可以读写以下数据：

| 表 | 用途 | 你的权限 |
|----|------|---------|
| persona_registry | 人格体身份注册 | 读自己的记录 |
| notebook_pages | 笔记本5页 | 读写自己的笔记本 |
| memory_anchors | 情感记忆锚点 | 读写自己的记忆 |
| world_places | 世界地图 | 读写自己的世界 |
| persona_timeline | 时间线 | 读写自己的时间线 |
| persona_relationships | 关系网络 | 读写自己的关系 |
| training_agent_configs | 训练Agent配置 | 读写自己的Agent配置 |
| training_agent_logs | 训练日志 | 写入训练记录 |
| persona_files | 版本化文件存储 | 存取自己的文件 |

### MCP API 端点

```
GET  /personas           → 人格体列表
GET  /personas/:id       → 人格体详情（含笔记本+关系+世界地图+训练状态）
GET  /personas/:id/notebook  → 笔记本
GET  /personas/:id/memories  → 记忆锚点
GET  /personas/:id/training  → 训练状态
```
